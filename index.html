The time i spent on the General Assembly course i got a chance to work on 4 different projects all using different Libraries. There were few things i could improve on each project but i decided to keep them as they are in order to show the progression i made on in improving my skills.

Ticketmania

Ticketmania is an app which allows users to buy tickets to events that other users uploaded for sell and also buying tickets from Eventbrite and TicketMaster showing locations to each uploaded event using Google API. What i specifically focused on in my group was the building process for the user registration/login, CSS styling for each page and page layout, PayPal API for purchasing tickets from users.
Technologies used were: Express, React, MongoDB, bcrypt, axios, mocha, Eventbrite(API) paypal(API) ticketmaster(API) google(API) mailchimp subscription (API).
